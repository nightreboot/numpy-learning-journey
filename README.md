# NumPy Learning Journey 🚀

A complete hands-on repository for learning **NumPy in Python** from basics to practical problem-solving.

This repository contains notebooks covering NumPy fundamentals, array operations, indexing, slicing, broadcasting, statistical calculations, and practice problems.

---

## 📌 Topics Covered

### 1. NumPy Basics

* Creating NumPy arrays
* Zeros, Ones, Random arrays
* Matrix creation
* Vector, Matrix, Tensor basics

### 2. Array Indexing & Slicing

* Vector indexing
* Matrix indexing
* Row & column slicing
* Multi-dimensional access

### 3. NumPy Operations

* Arithmetic operations
* Broadcasting
* Element-wise calculations
* Matrix operations

### 4. Statistical Operations

* Mean
* Percentiles
* IQR (Interquartile Range)
* Outlier detection

### 5. Practice Questions

* Student dataset analysis
* Matrix-based problem solving
* Sudoku matrix operations
* Real-world NumPy exercises
